# User Management System (CRUD)

A full-stack CRUD application built using:
- Node.js
- Express.js
- MySQL
- EJS
- Method-Override

Features:
- Create user
- Read users
- Update username with password verification
- Delete user with password verification
- RESTful routing

To run locally:
1. npm install
2. Create MySQL database named 'delta'
3. Run node index.js
